# 新机必装软件列表

> 本文记录了我在新电脑上安装的软件，既作为自己的备份参考，也分享给有需要的朋友。

## 导语

每次重装系统或换电脑，最让人头疼的就是重新安装和配置各种软件。为了避免每次都要回忆一遍装了什么、以什么顺序装，特意整理了这个软件清单。

这份列表是我根据自己的工作习惯整理的，主要涵盖了**日常通讯**、**开发工具**、**办公效率**、**系统工具**等类别。安装顺序我也调整到相对合理——有些基础组件需要先装，其他软件才能正常运行。

---

## 软件列表

### 1. [微信 (WeChat)](https://pc.weixin.qq.com/?lang=zh_CN)
- 无需解释的微信。

### 2. [Microsoft Office](https://account.microsoft.com/services)
- Word、Excel、PowerPoint 三件套，在微软账号的订阅清单里可以找回。

### 3. [.NET Desktop Runtime 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-8.0.28-windows-x64-installer?cid=getdotnetcore)
- 许多 Windows 应用程序需要的运行时库，后面安装其他软件的时候也会用到。

### 4. [飞书 (Feishu)](https://www.feishu.cn/download)
- 其实我日常上班公司并不会用飞书，但我觉得飞书多维表格非常好用，还很容易接到Agent中。我喜欢用飞书管理一些计划表格等等。

### 5. [Git](https://git-scm.com/)
- Git，必装。

### 6. [TortoiseGit](https://tortoisegit.org/download/)
- Git 的图形化界面，可以快速查log、编辑commit记录、进行rebase等。界面比较古朴，但非常好用。

### 7. [Visual Studio Code](https://code.visualstudio.com/Download)
- 我基本上一个 VS Code 打天下。

### 8. [Miniconda3](https://www.anaconda.com/download/success?reg=skipped)
- 相对 Anaconda 我更喜欢 miniconda，比较轻量。

### 9. [Node.js](https://nodejs.org/en/download)
- 我基本不进行 js 开发，但时常有些工具软件等需要 nodejs。

### 10. [Claude Code](https://claude.com/product/claude-code)
- 我比较喜欢 Claude Code CLI 做 vibe coding。

### 11. [CC Switch](https://github.com/farion1231/cc-switch/releases)
- CC Switch 可以快速切换各种 Agent 的配置，可以切换不同的模型、coding plan等。由于 CC 的接口经常不稳定，Coding Plan 又时不时429，我一般会有几个轮换。

### 12. [Xshell 和 Xftp](https://www.netsarang.com/en/free-for-home-school/)
- 强大的 SSH 终端模拟器和FTP/SFTP 文件传输客户端，远程服务器管理必备。

### 13. [Snipaste](https://www.snipaste.com/)
- 一键截图，还能贴到屏幕上，用法很多，效率提升神器。
